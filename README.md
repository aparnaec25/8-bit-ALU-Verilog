# 8-bit ALU Design using Verilog HDL

## Operations Implemented

- Addition
- Subtraction
- AND
- OR
- XOR
- NOT
- Left Shift
- Right Shift

## Tools Used

- Xilinx Vivado
- Verilog HDL

## Inputs

- A [7:0]
- B [7:0]
- SEL [2:0]

## Outputs

- Result [7:0]
- Carry
- Zero
